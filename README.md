### Installation

This system requires [Node.js](https://nodejs.org/) and [Ionic](https://ionicframework.com) to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd ionic-ShoppingApp
$ npm i
$ ionic serve --lab -c
```
